# Recordatorio-de-tareas



RD - App de tareas



Descripción:

Aplicación móvil para organizar tareas escolares.



Objetivo:

Mejorar la productividad de estudiantes.



Usuarios:

Estudiantes de secundaria.



Problema:

Olvidan tareas y fechas de entrega.



Funcionalidades:



Crear tareas

Notificaciones

Calendario



Requisitos funcionales:

Registro de usuario

Inicio de sesión

CRUD de tareas



Requisitos no funcionales:



Tiempo de carga menor a 2 segundos

Compatible con Android



Métricas:



100 usuarios activos en el primer mes
